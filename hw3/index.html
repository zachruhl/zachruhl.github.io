Homework #3: Weather Widget

Paris

It is 81 degrees outside.

Get Current Temperature

Can you modify weather.js to display the current weather conditions outside?

  1. 10 points: Show the current weather in Paris when the Get Current Temperature button is clicked.

    • Use the Console to play with the getWeather() function for a bit.
    • Sign up for a free API key from OpenWeatherMap.
    • Change the code to use your API key instead of mine.
    • Finish the code that's been started for you in weather.js to update the displayed temperature.
      Notice that the latitude and longitude for Paris have already been hardcoded for you, and the provided code will call the OpenWeatherMap API and retreive the results. However, the code isn't quite finished. Use your debugging skills to determine how to finish the task.
    • The temperature value should be rounded to the nearest integer.

  2. 6 points: Use your actual location to update the widget.

    • Use the browser's navigator.geolocation object to obtain the user's actual real-time latitude and longitude.
      You'll need to call the getCurrentPosition() function and provide a callback function in order to receive the user's latitude and longitude data.
    • When you're finished, both the temperature and city name should reflect the your location and current weather.

  3. 4 points: Update the weather icon in the widget. (See the hint at the bottom of weather.js)
